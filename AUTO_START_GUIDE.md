# ကွန်ပျူတာဖွင့်လျှင် ဆော့ဝဲလ် အလိုအလျောက် ပွင့်အောင် လုပ်နည်း

ဆိုင်ဖွင့်ချိန် ကွန်ပျူတာ Power ဖွင့်လိုက်သည်နှင့် POS စနစ် အလိုအလျောက် တက်လာစေရန် အောက်ပါအတိုင်း တစ်ကြိမ် လုပ်ဆောင်ပေးပါ။

1.  ကီးဘုတ်မှ `Windows Key` နှင့် `R` ကို တွဲနှိပ်ပါ။ (Run box ပွင့်လာပါမည်)
2.  `shell:startup` ဟု ရိုက်ထည့်ပြီး Enter ခေါက်ပါ။ (Startup Folder ပွင့်လာပါမည်)
3.  သင်၏ Project Folder ထဲရှိ `start_pos_lan.bat` ဖိုင်ကို Right Click နှိပ်ပြီး `Create Shortcut` လုပ်ပါ။
4.  ရလာသော Shortcut ဖိုင်လေးကို ခုနက ဖွင့်ထားသော Startup Folder ထဲသို့ ဆွဲထည့်လိုက်ပါ။

ဒါဆိုရင် ပြီးပါပြီ။ နောက်ရက်များတွင် ကွန်ပျူတာဖွင့်လိုက်တာနဲ့ POS စနစ် အလိုအလျောက် ပွင့်လာပြီး အသုံးပြုနိုင်ပါပြီ။

**သတိပြုရန်:**
အနက်ရောင် Screen လေးတစ်ခု ပေါ်နေပါလိမ့်မည်။ ၎င်းသည် Server အလုပ်လုပ်နေခြင်းဖြစ်သဖြင့် **မပိတ်ပါနှင့်**။ Minimize (အောက်ချ) လုပ်ထားနိုင်ပါသည်။
